# 放羊小游戏开发初稿

## 一、游戏定位

使用 Godot 4 制作一款 2D 俯视角像素风放羊小游戏。

第一版先完成一个能运行的场景：玩家从封面进入游戏，看到一块草地和一群羊。玩家能在草地上移动，羊会随机活动。完成这个基础后，再加入赶羊、羊圈和计数玩法。

## 二、第一版目标

### 阶段 1：封面和基础场景

- [ ] 创建 Godot 4 项目
- [ ] 设置 16:9 游戏窗口
- [ ] 制作游戏封面
- [ ] 添加“开始游戏”和“退出游戏”按钮
- [ ] 点击开始按钮后进入草地场景

- [ ] 放置树木、石头、灌木等装饰
- [ ] 在草地上放置 5 至 10 只羊
- [ ] 添加顶部信息栏，先显示羊的总数

### 阶段 2：基础操作

- [ ] 添加牧羊人角色
- [ ] 支持键盘方向键和 WASD 移动
- [ ] 给角色、树木和石头添加碰撞
- [ ] 让摄像机跟随玩家
- [ ] 让羊随机走动和停留

### 阶段 3：第一轮玩法

- [ ] 玩家靠近时，羊向远处移动
- [ ] 添加羊圈区域

## 三、场景规划

### 封面场景 `TitleScreen.tscn`

```text
TitleScreen（Control）
├─ Background（TextureRect）
├─ Title（Label 或 TextureRect）
├─ StartButton（Button）
└─ QuitButton（Button）
```

封面先使用一张草地背景或游戏场景截图。游戏画面完成后，再制作正式封面。

### 游戏场景 `Game.tscn`

```text
Game（Node2D）
├─ Ground（TileMapLayer）
├─ Decorations（Node2D）
├─ SheepGroup（Node2D）
├─ Player（CharacterBody2D）
├─ SheepPen（Area2D）
├─ Camera2D
└─ UI（CanvasLayer）
   ├─ TopBar
   └─ ResultPanel
```

### 羊场景 `Sheep.tscn`

```text
Sheep（CharacterBody2D）
├─ Sprite2D 或 AnimatedSprite2D
├─ CollisionShape2D
└─ Timer
```

每只羊都使用同一个 `Sheep.tscn`，游戏场景可以重复创建多只羊。

## 四、技术选择

- 游戏引擎：Godot 4 稳定版
- 开发类型：2D
- 脚本语言：GDScript
- 地图制作：TileMapLayer
- 基础素材：Tiny Swords
- 目标平台：Windows 电脑
- 建议基础分辨率：1280 × 720

像素素材需要关闭纹理过滤，避免图片发糊。项目开始后统一设置为 Nearest，并使用整数缩放。

## 五、素材清单

### 现在需要准备

- [ ] 下载完整的 Tiny Swords 资源包
- [ ] 保留资源包的授权说明文件
- [ ] 准备一套俯视角像素羊素材

Tiny Swords 官方页面：<https://pixelfrog-assets.itch.io/tiny-swords>

先下载完整资源包，不用逐张筛选。开始搭建项目时，我们会从中选择草地、树木、石头、灌木、围栏和 UI 素材。

### 可以稍后准备

- [ ] 牧羊人角色素材
- [ ] 羊的行走和吃草动画
- [ ] 游戏标题字样或 Logo
- [ ] 按钮点击、羊叫和背景音乐
- [ ] 正式封面图片

如果暂时找不到合适的羊，可以先用白色圆形或临时图片代替。先验证玩法，之后替换图片不会影响脚本结构。

## 七、首个可运行版本的完成标准

1. 启动游戏后显示封面。
2. 点击“开始游戏”进入草地。
3. 草地上显示树木、石头和至少 5 只羊。
4. 玩家可以移动，且不会穿过障碍物。
5. 羊会在草地范围内随机活动。
6. 游戏可以退出，并且运行时没有报错。
