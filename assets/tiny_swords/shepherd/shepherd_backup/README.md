# Shepherd Backup Sprites

Reserve sprite pack for the black-haired shepherd girl with round glasses and a cream-white quilted jacket.

All PNG files have transparent backgrounds and 128x128 frame cells. Horizontal strip width divided by 128 is the frame count.

## Animation Map

| Animation | Direction | Frames | Loop |
| --- | --- | ---: | --- |
| `idle` | down, up, left, right | 1 each | yes |
| `walk` | down, up, left, right | 6 each | yes |
| `run` | down, up, left, right | 7 each | yes |
| `lie_down` | side | 6 | no |
| `rest` | side | 6 | yes |
| `whistle` | down-right | 6 | no |
| `tend` | right | 6 | no |
| `tend_lamb` | right | 6 | no |
| `carry_lamb` | down-right | 4 | no |

The original GPT Image 2 sheets, alpha intermediates, prompts, and packing script are kept in `work/imagegen/shepherd/`.
